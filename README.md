# Python.sh-Expense-Task-3
https://roadmap.sh/projects/expense-tracker

Cara Menggunakan

1. Tambah Pengeluaran:
python expense_tracker.py add --description "Makan Siang" --amount 20

2. Lihat Semua:
python expense_tracker.py list

3. Hapus:
python expense_tracker.py delete --id 1

4. Ringkasan Total:
python expense_tracker.py summary

5. Ringkasan Per Bulan:
python expense_tracker.py summary --month 5 (untuk bulan Mei)
